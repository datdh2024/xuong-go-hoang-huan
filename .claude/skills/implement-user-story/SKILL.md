description: Use this skill when you need to implement a user story.
---

- Apply /test-driven-development skill, wait until user confirm each steps.
- Require user evaluate task completion
- After user confirm, update all related files (user-flows, tests, etc.). Update status, note to Jira
- Create branch (if not exist) for parent user story, commit and push code to branch
